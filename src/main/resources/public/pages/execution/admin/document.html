算法管理

添加算法信息

  • 标签不要大写，即便是 doctype 标签。

通过excel文件添加算法信息

  • 1.点击添加算法信息按钮。
  • 2.选择要上次的算法信息文件（仅支持.xsls、.xls文件）。
  • 3.点击确定。
  • 4.算法信息添加完成。

修改算法信息

  • 1.选择要修改算法信息的算法。
  • 2.填写修改信息。
  • 3.点击确认改修。
  • 4.完成算法信息修改。

删除算法信息

  • 1.选择要删除的算法。
  • 2.确认删除算法信息。
  • 3.点击确认。
  • 4.完成算法删除。

业务管理

添加用户信息

  • 1.点击添加用户按钮。
  • 2.填写添加用户的信息。
  • 3.点击确认添加用户。

修改用户信息

  • 1.选择点击要修改信息的用户。
  • 2.填写要修改的用户信息。（管理员不可修改用户密码）
  • 3.点击确认用户信息修改。

删除用户信息

  • 1.点击选择要删除的用户。
  • 2.点击确认要删除的用户信息。
  • 3.删除结果信息回显。

添加管理员账户信息

  • 1.点击点击管理员账户。
  • 2.添加管理员账户基本信息。
  • 3.点击添加管理员账户。
  • 4.添加管理员账户信息回显。
  • tips:不能添加和删除用户名为"root"的管理员账户，该管理员账户只允许密码的修改，账户名不可修改。添加管理员账户时，每个管理员账户名不可重复！

修改管理员信息

  • 1.点击要修改的管理员账户。
  • 2.填写修改管理员账户信息。
  • 3.点击确实修改管理员账户信息。
  • 4.修改信息结果回显。
